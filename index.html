Minha loja de raçoẽs online

O projeto é um aplicativo criado com HTML e CSS, destinado a exibir uma loja virtual de raçoẽs para caẽs. O objetivo é criar um ambiente virtual onde seja possível compartilhar seus produtos, fornecendo uma descrição breve de cada um, incluindo o fornecedor, o ano de publicação e uma opção de compra.

A estrutura do aplicativo é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo, seguido por uma divisão principal:(.container), cada ração é separado organizadamente em uma lista (.livros), separa as raçoẽs por raça e tamanho dos caẽs, (.livro), mostrando uma imagem do produto, o preço e a previsão de entrega. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um aplicativo legal e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do aplicativo, incentivando a compra das raçoẽs.
